# Feedback Template

Structure for post-class teacher feedback. Replace ALL bracketed placeholders with actual class data.

## Format

```
📬 课后小结 · Lesson [N]

[日期]

🌱 整体印象
[2-3 sentences. Mention student count. Warm, positive, celebrate effort. 
Use "我们" not "你们". Name the specific thing students did well.]

💪 语法 · [specific focus area]
[3-4 sentences. Reference actual sentences students said in class. 
Give direct advice, not preachy. Connect to real-world usage.
Include the actual class example sentences as a highlighted block.
End with actionable takeaway.]

📝 本周作业 · Homework

1 [🎬/🎙️/📖] [Task name]（[书面/录音]）
[Description. Include ACTUAL class example sentences in a reference block.]
[If recording: include template with [name] placeholders.]

2 [emoji] [Task name]（录音）
[Description + template]

3 [emoji] [Task name]（录音）
[Description + sentence patterns from courseware]

4 🔁 录完后回听
听自己的录音，找到一处你想改善的地方（只改一处），再录一次。
两个版本都保留，下次课带来对比。

🔭 下节课预告
[One-line preview of next class topic]
```

## Tone Checklist
- [ ] Warm, not preachy
- [ ] Uses "我们" perspective
- [ ] Example sentences are FROM actual courseware (never invented)
- [ ] Homework templates match what was taught
- [ ] Pronunciation: specific sounds named, not generic "work on pronunciation"
- [ ] Grammar: connects to real usage, not abstract rules
- [ ] Homework section visually distinct (purple highlight in HTML)
