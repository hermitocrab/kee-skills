# MemPalace Room Catalog

Wing: `workspace` | Palace: `~/.mempalace/palace`

## Room Map

| Room | Contents | Scope when to use |
|---|---|---|
| ielts_website | Web app pages, CSS, JS, SQL | Website questions, Supabase, whiteboard |
| ielts_toolkit | Speaking mindmap, toolkit data | Speaking exercises, IELTS materials |
| general | Root-level docs (SOUL, USER, AGENTS, etc.) | Identity, rules, workspace config |
| bin | Build scripts, generated code | Script debugging, generated artifacts |
| memory | Daily notes, MEMORY.md | Past decisions, logs, incidents |
| rkrk_landing | rkrk landing page project | rkrk.io design/development |
| editorial_slide_designer | Slide design skill | Presentation design questions |
| agents | Agent prompt files | Agent behavior, prompt architecture |
| dog_restaurant_deck | Business plan + interactive deck | Woof & Dine project |
| tedx_web_slides | Web-based TEDx slides | TEDx web presentation |
| design | Assets, images | Visual assets, logos |
| photo_skill | Photo skill files | Photo manipulation questions |
| avatars | Identity assets | Avatar/icon questions |
| state | State files | Runtime state questions |

## Discovery

Run `~/Library/Python/3.9/bin/mempalace status` to refresh room counts. Rooms are auto-detected from `mempalace.yaml` in the workspace root.
