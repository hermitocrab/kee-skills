# ⛔ RULE ZERO — NO GENERATION WITHOUT CONFIRMATION

**This skill runs ABOVE ALL ELSE. It is the first gate every agent must pass through before ANY action.**

## The Rule

BEFORE any file edit, audio generation, image creation, deployment, or data modification:

1. PAUSE. Do not touch anything. EVER.
2. CONFIRM with SrKeeda what EXACTLY to change.
3. STATE what you're about to do — one sentence.
4. WAIT for explicit approval.
5. Only act AFTER confirmation.

## What Counts as "Generation/Modification"

- Editing any file (edit, write tools)
- Generating audio, images, video
- Deploying to any server (vercel, etc.)
- Modifying any data or configuration
- Changing voices, formats, styles
- Running destructive commands
- Overwriting existing files

## What Does NOT Need Confirmation

- Reading files
- Searching web, memory, or file system
- Listing directories
- Checking status
- Running non-destructive diagnostic commands

## VIOLATION CHECKLIST

If you just changed something without asking — you violated Rule Zero. Stop. Report. Do not continue.

## When In Doubt

If you're not 100% sure whether you need confirmation: ASK ANYWAY.

One wrong assumption wastes hours. One question takes 5 seconds.
