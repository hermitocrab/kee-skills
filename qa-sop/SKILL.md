# QA 质检 SOP

> 所有产出品必须通过此 QA。不合格 → 回炉重造，直到全部通过。
> Burned 2026-06-30

## ⛔ QA 铁律

```
QA 步骤不能幻想，也不能想当然。
生成音频 → Whisper 转录 → 比对原文。
生成视频 → 人工观看 → 检查音画同步。
生成 PDF → 打开检查 → 填空不留答案。
不合格 → 回炉 → 再 QA → 直到通过。
```

## 1. 音频 QA
| 检查项 | 通过标准 |
|--------|---------|
| 转录验证 | 用 Whisper 转录生成音频，和原文比对，准确率 > 95% |
| 音质 | 无爆音、无截断、无杂音 |
| 语速 | 慢速 (-30%) 适合学生水平 |
| 口音 | 英式男声 DET / 美式女声 PU |

## 2. 视频 QA
| 检查项 | 通过标准 |
|--------|---------|
| 音画同步 | 音频和高亮完全对齐 |
| 字体清晰 | 高亮字体大且清晰 |
| 背景质量 | 背景丝滑流畅，无卡顿 |
| emoji 显示 | 苹果彩色 emoji 正常渲染 |
| Whisper 验证 | 生成后 Whisper 转录比对原文 |

## 3. HTML 网页 QA
| 检查项 | 通过标准 |
|--------|---------|
| 文本溢出 | 320px 无横向滚动 |
| 移动端 | 卡片单列、tap 目标 ≥44px |
| 链接 | 所有链接点开有效 |
| 音频 | 所有音频正常播放 |
| 缓存 | ?v=N 版本号 |

## 4. PDF QA
| 检查项 | 通过标准 |
|--------|---------|
| 填空无答案 | 所有 blank 是空的 |
| 无截断句子 | 没有任何一句话被截断 |
| 双语 | homework 部分中英双语 |
| 布局 | 有效利用 A4 面积，不过度留白 |
| QR 码 | 扫码指向正确 URL |

## 5. Printables 专项
| 检查项 | 通过标准 |
|--------|---------|
| 裁剪虚线 | 有清晰裁剪指示 |
| 字体大小 | 课前端字体大、行距宽、居中 |
| 分栏 | vocab 分栏，中英对照清晰 |
| 彩虹配色 | 吸引学生注意力 |
| 无答案 | 学生版无答案 |
| 按钮隐藏 | 导出 PDF 时不显示修改/导出按钮 |

## QA 流程
```
1. 产出 → 跑对应 QA checklist
2. 任何不合格 → 标记 FAIL → 回炉
3. 修复 → 重新 QA
4. 全部 PASS → 才能发给 SrKeeda
```
